# @mdmend/darwin-arm64

Platform-specific binary for [mdmend](https://www.npmjs.com/package/mdmend).

This package is installed automatically as an optionalDependency of `mdmend`.
You do not need to install it directly.

## Usage

Install the main package:

```bash
npm install -g mdmend
```
